# IPPO 仕様メモ

> 動けない時に、少しずつ一歩を刻む。
> AI がタスクを小さく分けて、次の一歩を出してくれるアプリ。

このファイルは「今どこまで作ったか」「次に何をするか」を思い出すためのメモです。
しんどい日に開いても、ここを見れば再開できます。

---

## 今できること（この時点で完成している部分）

AI なしで、もう毎日使える状態です。

- **今日タブ**：毎日の習慣が自動で並ぶ＋今日やることを選ぶ → 手順を手で足す → ✓ でチェック →「できた」
  - まだ終わっていない最初の手順に「次の一歩」の印が付く
- **管理タブ**：カテゴリ・タスク・習慣（毎日）を登録・削除できる
- **ストックタブ**：買い物 / 視聴 / 行きたい場所 / メモ の4つのリスト
- **カレンダータブ**：月表示。できた数が見える。日付を押すとその日の「できたこと」一覧。メモも足せる
- データはブラウザ内に保存（サーバー不要）。「できたこと」は元タスクを消しても残る

### Any.do からの取り込み

`src/seed.ts` に初回取り込みデータがある。`main.tsx` が
`ippo:imported:anydo-v1` フラグを見て「まだなら一度だけ」取り込む。
取り込んだ内容はアプリ上から自由に編集・削除できる。

- 裁縫タスク5件＝カテゴリ「裁縫」・今日に追加
- [毎日]6件＝習慣
- Someday 5件＝タスク（今日には入れない）
- 買い物5件・視聴1件＝ストック
- 川口プラネタリウム／皮膚科は今日のタスク扱い（行きたい場所に移したければアプリで変更）

## データの形（src/types.ts）

- Category … 分類
- Task … 一度きりのやること
- Habit … 繰り返すもの（※型だけ用意。中身は次回）
- Step … 手順。parentId で「どのタスクの手順か」を表す。手順の手順（再分解）も表現可能
- TodayItem … 今日やることに選んだもの
- DoneLog … できたことの記録（title をコピーして持つので消えない）

保存と操作は `src/store.ts` に集約。画面は `useStore()` で最新データを読むだけ。

### AI を入れるときのメモ（大事）

鬱の人が使うので、プロンプトで次を必ず指定する：

- 1ステップを **極端に小さく**（「立ち上がる」「コップを持つ」レベルまで OK）
- **負担の軽い、責めない言葉**で
- 一度に出すのは **3〜5個** まで（多いと圧になる）
