## 今できること（この時点で完成している部分）

AI なしで、もう毎日使える状態です。

- **管理タブ**：カテゴリとタスクを登録・削除できる
- **今日タブ**：今日やることを選ぶ → 手順を手で足す → ✓ でチェック →「できた」
  - まだ終わっていない最初の手順に「次の一歩」の印が付く
- **カレンダータブ**：月表示。できた数が見える。日付を押すとその日の「できたこと」一覧。メモも足せる
- データはブラウザ内に保存（サーバー不要）。「できたこと」は元タスクを消しても残る

## データの形（src/types.ts）

- Category … 分類
- Task … 一度きりのやること
- Habit … 繰り返すもの（※型だけ用意。中身は次回）
- Step … 手順。parentId で「どのタスクの手順か」を表す。手順の手順（再分解）も表現可能
- TodayItem … 今日やることに選んだもの
- DoneLog … できたことの記録（title をコピーして持つので消えない）

保存と操作は `src/store.ts` に集約。画面は `useStore()` で最新データを読むだけ。
